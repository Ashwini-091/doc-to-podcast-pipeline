SETUP (5 MINUTES)
================================================================================

1. GET FREE API KEY (1 minute)
   Visit: https://console.groq.com
   - Sign up (free, no credit card!)
   - Copy your API key (looks like: gsk-xxxxx...)

2. INSTALL DEPENDENCIES (1 minute)
   $ pip install -r requirements.txt

3. SET API KEY (1 minute)

   Windows (Command Prompt):
   $ set GROQ_API_KEY=gsk-your-key-here

   Windows (PowerShell):
   $ $env:GROQ_API_KEY="gsk-your-key-here"

   macOS/Linux:
   $ export GROQ_API_KEY="gsk-your-key-here"

4. VERIFY INSTALLATION (1 minute)
   $ python setup_and_verify.py

   Expected output:
   ✅ All checks passed!

5. RUN THE APP (1 minute)
   $ streamlit run app.py

   Browser opens automatically.

================================================================================
TROUBLESHOOTING
================================================================================

❌ "GROQ_API_KEY not set"
→ Set the environment variable in your terminal:
  export GROQ_API_KEY="gsk-..."

❌ "No module named 'groq'"
→ Reinstall dependencies:
  pip install -r requirements.txt --force-reinstall

❌ "Too many requests"
→ Free tier has rate limits. Wait a moment and try again.

❌ "Model not found"
→ Check your API key is correct at https://console.groq.com
